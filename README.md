rptree
======

show running process tree and focus it's environ changes.

# Compile and run example

➜  rptree git:(master) ✗ cd build
➜  build git:(master) ✗ cmake ..
➜  build git:(master) ✗ make
➜  build git:(master) ✗ ./rptree -- sh ../example/1.sh
